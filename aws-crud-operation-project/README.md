\# AWS CRUD Operation Project



\## Project Overview

This project demonstrates CRUD (Create, Read, Update, Delete) operations using Python Flask connected with AWS RDS MySQL database.



\## Technologies Used

\- AWS EC2

\- AWS RDS

\- Python Flask

\- MySQL

\- HTML

\- CSS

\- Linux



\## Features

\- Add Employee

\- View Employee

\- Update Employee

\- Delete Employee

\- Database Integration with AWS RDS



\## AWS Services Used

\- EC2

\- RDS

\- Security Groups

\- IAM



\## Steps Performed

1\. Created EC2 instance

2\. Installed Python and Flask

3\. Configured AWS RDS MySQL

4\. Connected EC2 with RDS

5\. Developed CRUD application

6\. Hosted application on EC2
